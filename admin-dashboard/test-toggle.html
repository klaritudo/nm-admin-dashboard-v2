슬롯 설정 페이지 토글 테스트

브라우저에서 다음 단계를 따라하세요:

  1. 브라우저에서 http://localhost:5173 접속
  2. 로그인 (필요한 경우)
  3. 사이드바에서 "게임설정" > "슬롯 게임설정" 클릭
  4. 또는 직접 http://localhost:5173/game-settings/slot 접속
  5. 개발자 도구(F12) 열고 Console 탭 확인
  6. 페이지 로드 시 나오는 로그 확인
  7. 테이블 헤더에서 "연속번호/페이지별번호" 토글 버튼 클릭
  8. 토글 클릭 시 Console에 나오는 로그 확인
  9. 번호 컬럼(첫 번째 컬럼)의 UI 변화 확인

예상되는 로그

페이지 로드 시:

CellRenderer 호출: {columnId: "no", columnType: "number", sequentialPageNumbers: true, page: 0, rowIndex: 0}
CellRenderer 번호 컬럼 렌더링 [행0]: {sequentialPageNumbers: true, page: 0, rowsPerPage: 10, rowIndex: 0, displayNumber: 1, columnId: "no", rowData: "..."}

토글 클릭 시:

슬롯 페이지 번호 모드 토글: false (연속번호 -> 페이지별번호로 변경)
토글 후 sequentialPageNumbers 값: false
SlotSettingPage finalColumns: [{id: "no", type: "number", label: "No."}, ...]
CellRenderer 번호 컬럼 렌더링 [행0]: {sequentialPageNumbers: false, page: 0, rowsPerPage: 10, rowIndex: 0, displayNumber: 1, columnId: "no", rowData: "..."}
